Annual Self-Classification: Encryption

App name: Red Cocoa

Bundle ID: com.redcocoa.app

This app uses only encryption that is exempt from the U.S. Export Administration Regulations (EAR) and equivalent requirements. Specifically:

  • The app uses encryption solely for standard data in transit (e.g., HTTPS/TLS) via Apple’s operating system and standard system frameworks.
  • The app does not use any proprietary encryption algorithms.
  • The app does not implement or use any standard encryption algorithms beyond what is provided by or accessed through Apple’s iOS.

Therefore, no export classification (CCATS) or other encryption documentation is required for this app. The app’s Info.plist includes ITSAppUsesNonExemptEncryption set to false.
