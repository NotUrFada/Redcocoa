# Red Cocoa – Legal (GitHub Pages)

This folder is used for **GitHub Pages** so the privacy policy has a public URL for App Store Connect.

**Privacy Policy URL (after enabling Pages):**  
**https://NotUrFada.github.io/Redcocoa/**

## Enable GitHub Pages

1. Push this repo to GitHub (if not already).
2. On GitHub: **Settings** → **Pages**.
3. Under **Build and deployment** → **Source**: choose **Deploy from a branch**.
4. **Branch**: `main` (or `master`) → **Folder**: `/docs` → **Save**.
5. Wait 1–2 minutes. Your site will be at **https://NotUrFada.github.io/Redcocoa/**
6. In App Store Connect, set **Privacy Policy URL** to **https://NotUrFada.github.io/Redcocoa/** and save.
